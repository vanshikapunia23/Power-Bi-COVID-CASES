# Power-Bi-COVID-CASES
Data visualization project using Power BI to analyze COVID-19 trends, case growth, recovery rates, and mortality from 2020 to 2022.
📊 COVID-19 Data Analysis Dashboard (2020–2022)
🔍 Project Overview

This project presents a comprehensive analysis of global COVID-19 data from 2020 to 2022 using an interactive Power BI dashboard. The goal of this project is to explore trends, patterns, and key insights related to the spread and impact of COVID-19 across different regions and time periods.

🎯 Objectives
To analyze the growth of COVID-19 cases over time
To compare confirmed cases, recoveries, and deaths
To identify trends and patterns across different countries/regions
To provide a clear and interactive data visualization for better understanding
📈 Key Features
Interactive dashboard with dynamic filters and slicers
Time-based analysis (yearly/monthly trends from 2020–2022)
Comparison of total confirmed cases, recovered cases, and deaths
Region-wise and country-wise analysis
Visual insights using charts, graphs, and maps
🛠️ Tools & Technologies Used
Power BI (Data Visualization)
Microsoft Excel / CSV (Data Source)
Data Cleaning & Transformation (Power Query)
📊 Insights Gained
Rapid increase in COVID-19 cases during peak waves
Variation in recovery and death rates across regions
Identification of high-impact countries and time periods
Clear visualization of how the pandemic evolved over time
📁 Project File
COVID_Analysis.pbiX

🚀 Conclusion

This project demonstrates how data visualization tools like Power BI can be used to transform raw data into meaningful insights. It helps in understanding the global impact of COVID-19 and supports data-driven decision-making.
